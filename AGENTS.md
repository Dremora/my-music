# AI Agent Rules - My Music

## Tooling

Use pnpm. Never use npm or npx.

Use jj for version control. Do not use git commands in this repository unless the
user explicitly asks for them.

If `jj` is installed, read `docs/agents/jj.md` before running any `jj` command
or editing files. Follow that workflow for status, diffs, revisions, and
workspace changes.

Always use exact versions in `package.json`. Do not add `^` or `~` prefixes.
